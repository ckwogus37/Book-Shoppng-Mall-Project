# Book-Shoppng-Mall-Project
프로그래머스 웹 풀 사이클 데브코스 프로젝트 (스프린트 2)

# 도서 구매 쇼핑몰 구현
## 기술 스택
1. JavaScript
2. Express.js (dotenv, express-validator, jsonwebtoken, mysql2)
3. MariaDB (with Docker)

## 기능
### 회원 관련 기능
    1. 회원가입 
    2. 로그인
    3. 비밀번호 초기화

### 도서 관련 기능
    1. 전체 도서 조회
    2. 개별 도서 조회
    3. 카테고리 별 도서 조회
    4. 좋아요 기능

### 주문 기능
    1. 장바구니 기능
    2. 결제 기능
    3. 결제 내역 조회
